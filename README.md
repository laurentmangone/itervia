# Itervia

**Itervia** is a desktop application for planning bicycle and hiking routes. It is built with **Tauri v2**, **React**, **TypeScript**, **MapLibre GL JS**, and the **OpenRouteService (ORS)** API.

---

## Features

- **Map display** using MapLibre GL with CartoDB Positron basemap (no API key required).
- **Route creation**: click on the map to add points, drag to reorder, insert between existing points, and delete.
- **GPX import / export** preserving altitude data.
- **Elevation profile** rendered directly on a canvas (no external chart library).
- **SQLite persistence** via `tauri-plugin-sql` – routes, points, and user preferences are stored locally.
- **User preferences**: store the ORS API key encrypted with AES‑256‑GCM in Rust; preferences are saved in SQLite.
- **Performance optimisations**: batch inserts, selective Zustand selectors, AbortController on network calls, code‑splitting for MapLibre.
- **Unit tests** with Vitest (39 passing tests) covering routing logic, store actions and utility functions.
- **Distribution**: macOS `.app` and `.dmg` bundles generated by Tauri.

## Architecture

```
src/
 ├─ components/            # UI components (MapView, RouteEditor, SettingsDialog…)
 ├─ hooks/                 # Custom React hooks
 ├─ services/              # Business logic – routing, GPX handling, ORS calls
 ├─ store/                 # Zustand stores (routeStore, preferencesStore)
 ├─ types/                 # TypeScript type definitions
 ├─ App.tsx                # Application entry point
 └─ index.html & style files

tsrc-tauri/
 ├─ src/
 │   ├─ lib.rs            # Rust bridge – ORS request, encryption
 │   └─ crypto.rs         # AES‑256‑GCM helpers
 └─ Cargo.toml            # Cargo configuration
```

## Getting Started

```bash
# Clone the repository
git clone https://github.com/laurentmangone/itervia.git
cd itervia

# Install Node dependencies
npm install

# Build and run the dev version
npm run tauri dev   # uses --no-watch to avoid file‑watcher crash
```

### Production build

```bash
npm run tauri build   # produces .app and .dmg in src-tauri/target/release/bundle/
```

## Configuration

- **ORS API key**: open Settings (gear icon) → paste your OpenRouteService key. The key is encrypted on the Rust side and stored in the `preferences` SQLite table.
- **Map tiles**: CartoDB Positron via `https://basemaps.cartocdn.com/gl/positron-gl-style/style.json`.
- **Geocoding**: Nominatim (`https://nominatim.openstreetmap.org/search`).

---

## Development notes

- The routing function `calculateRoute` now receives only coordinates and a profile; abort handling is managed internally where needed.
- Unused `AbortController` instances were removed to keep the TypeScript build clean.
- Chunk size warnings from Vite are mitigated by the existing manual code‑splitting configuration.
- Rust code (`src-tauri/src/crypto.rs`) implements AES‑256‑GCM encryption with a key derived from the host name.
